# 产品介绍

Cloudpods AI云 面向大语言模型（LLM）及各类 AI 容器应用的统一管理与运维，帮助企业在同一平台完成 AI 工作负载的部署、扩缩与监控。

## 定位说明

纳管、调度与生命周期管理 LLM 及 AI 容器应用，与私有云/多云协同。

## 功能概览

- AI 容器应用的部署与编排
- 大语言模型等服务的资源调度与监控
- 与现有 Cloudpods 资源与权限体系集成

## 支持的应用

平台支持以下典型 AI 应用：

- **Ollama**：本地 LLM 轻量方案。
- **vLLM**：高吞吐、低延迟推理服务。
- **OpenClaw**：开源自托管的个人智能体助手。
- **ComfyUI**：图像生成与工作流应用。

控制台「人工智能」下管理应用、推理、镜像；详见 [用户手册](../guides)。
