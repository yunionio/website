# 应用模板

应用模板用于定义 AI 应用实例（如 OpenClaw、Dify、ComfyUI）的 CPU/内存/GPU/存储配置，创建实例时按模板快速复用资源规格。

## 核心概念

控制台入口为 **人工智能 → 应用 → 应用模板**。创建应用实例时选择模板，即可获得预设的资源配置；当资源不足或约束不满足时，实例会调度失败。

常用配置维度包括：

- **类型**：对应AI应用实例的类型。
- **镜像**：关联对应服务的容器镜像。
- **CPU**：影响请求处理与并发能力。
- **内存**：影响应用进程与缓存稳定性；不足易发生 OOM 或重启。
- **数据盘**：用于数据的持久化存储，会根据。
- **GPU型号**：图像类应用（如 ComfyUI）依赖 GPU 资源。
- **端口映射**：从宿主机映射容器内服务的端口。
