# AI镜像

运行 AI 应用的容器镜像。控制台在 **人工智能 → 镜像** 管理；创建模板时选择镜像，LLM 类型需与模板一致。

## 镜像来源

- **平台预置**：默认镜像，快速部署。
- **公共仓库**：可联网时拉取。
- **私有仓库**：生产与合规场景。

## 兼容性

- **GPU**：镜像需与节点 NVIDIA 驱动/CUDA 兼容，见 [NVIDIA/CUDA](../getting-started/setup-nvidia-cuda)。
- **升级**：评估运行时依赖（推理引擎、Python/系统库等）兼容性。

## 常见问题

### 镜像拉取失败

检查网络与 DNS；私有仓库检查鉴权与权限。
