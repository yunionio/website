# AI规格

AI 规格用于定义 AI 工作负载（如推理服务、模型服务）所采用的算力与资源配置，通常包括 CPU、内存、GPU 类型与数量等维度。平台根据规格进行调度与资源分配。

## 规格说明

不同 AI 应用对规格的要求不同：依赖 GPU 的应用（如 Ollama、vLLM、ComfyUI）需选择带 GPU 的规格；不依赖 GPU 的应用（如 OpenClaw、Dify）可使用仅含 CPU 与内存的规格。具体可选规格项与使用方式将在控制台与后续文档中补充。

## 使用说明

在创建或部署 AI 应用时，在对应界面选择或填写所需 AI 规格即可。更多操作步骤将随版本迭代完善。
