# 用户手册

围绕 AI 应用、推理、模板、模型库、镜像的概念与使用说明。

## 阅读指南

- 部署见 [快速开始](../getting-started/quickstart)。
- GPU部署见 [NVIDIA/CUDA](../getting-started/setup-nvidia-cuda)。
- 按 [AI应用](./llm-app/)（含 [应用模板](./llm-app/template)）→ [AI推理](./llm-inference/)（含 [推理模板](./llm-inference/template) / [AI模型库](./llm-inference/model-library)）→ [AI镜像](./llm-image) 阅读。

## 控制台菜单与功能对照

进入 **人工智能** 后分为 **应用**、**推理**、**镜像**：

- **应用**：应用实例（Dify/OpenClaw/ComfyUI）、应用模板。
- **推理**：推理实例（Ollama）、推理模板、推理模型库。<!-- 推理实例（Ollama/vLLM）、推理模板、推理模型库。 -->
- **镜像**：在「人工智能」→「镜像」管理；创建模板时选择。

## 名词约定

- **模板**：运行实例所需的 CPU/内存/GPU 等资源配置。
- **模型库**：可复用模型资源，供推理应用使用。
- **镜像**：实例运行环境的容器镜像，决定功能与版本。

## 典型关系

实例 = 镜像 + 规格 +（可选）模型。
