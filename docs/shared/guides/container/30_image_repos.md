# 镜像仓库

平台提供镜像仓库的管理，可以注册容器镜像仓库（repository）。

虽然容器主机可以使用任意容器镜像，并不局限于注册的镜像仓库的镜像，但注册容器镜像仓库提供以下便利：

1. 镜像选择

如果镜像仓库支持列表API，则平台会获取镜像仓库中所有镜像和TAG。在使用web控制台创建容器主机时，可以通过图形化界面浏览仓库，选择镜像和TAG。

2. 身份认证

如果镜像仓库需要认证才能访问，则必须注册该镜像仓库，并保存镜像仓库的用户名和密码。在容器主机使用镜像时，会自动匹配该镜像对应的镜像仓库，如果该仓库访问需要认证信息，则会自动从镜像仓库获取认证信息，实现镜像的认证下载。
