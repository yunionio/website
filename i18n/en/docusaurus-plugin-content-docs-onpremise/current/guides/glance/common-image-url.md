# Common System Images

Display download links for common images.

## Community Images

Starting from 3.11, the platform has added "Community Images" functionality in the frontend system image console, integrating common publicly downloadable operating system images. Users can directly click to import. Integrated image information is maintained [here](https://github.com/yunionio/website/blob/master/static/openimages.yaml). If you find download links are invalid, or wish to add new images, you can submit a PR to this document.
