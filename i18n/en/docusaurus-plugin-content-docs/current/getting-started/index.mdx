# Getting Started

Please select the Cloudpods product to deploy according to your business scenarios:

- Private Cloud: Private cloud deployment includes the function of managing local IDC infrastructure and is a lightweight implementation of private cloud, including virtual machines, storage, network, GPU cards, and bare metal functions.
    - [Quick Start: Private Cloud Deployment](./onpremise/quickstart-virt)
- Multi-Cloud Management: Multi-cloud management deployment includes the functionality of managing VMware, private cloud, and public cloud to help enterprises manage heterogeneous IT infrastructure resources.
    - [Quick Start: Multi-cloud Management Deployment using ocboot](./cmp/quickstart-ocboot)
    - [Quick Start: Multi-cloud Management Deployment using docker compose](./cmp/quickstart-docker-compose)
    - [Quick Start: Multi-cloud Management Deployment using helm on Kubernetes](./cmp/quickstart-k8s-helm)
- Hybrid Cloud: Hybrid cloud deployment includes the functionality of lightweight private cloud and multi-cloud management.
    - [Quick Start: Hybrid Cloud Deployment](./full/quickstart-full)
